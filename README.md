# ツイタマモドキ (TwitamaModoki)

ツイタマの操作性を目指したX（旧Twitter）拡張機能

## 概要

ツイタマモドキは、X（旧Twitter）の複数のページを同時にカラムとして表示し、モバイルデバイスで効率的に閲覧できるChrome拡張機能です。

Android や iOS で拡張機能が使えるブラウザにインストールしてください。PC での使用は想定されていません。

動作確認は Kiwi Browser で行っています。

### 注意
まだまだバグがあります。

## 主な機能

### ✅ 実装済み

- **カラムベースのUI**
  - 複数のXページを同時に表示
  - ホーム、リスト、検索、DM、通知、Exploreに対応
  - 設定画面にてカラムの追加・削除・並び替え

- **スワイプ操作**
  - 横スワイプでカラム切り替え
  - 画像表示時、下スワイプで画像を閉じる

- **タブバー**
  - 主要ページのURLを自動保存し、ブラウザを閉じても前回の状態を復元
  - リスト名・コミュニティ名をタブに表示（長い名前は省略表示）

- **表示設定**
  - ポスト本文のフォントサイズ調整（80%〜150%）

- **スクロール補助**
  - フローティングスクロールボタン（上下）
  - 左下/右下の配置を設定で切り替え可能

- **レート制限表示**
  - X API のレート制限情報をリアルタイム表示
  - カラムの種類に応じて関連する制限のみ表示
  - 投稿アイコンタップで投稿画面に遷移

- **UI最適化**
  - 広告（プロモーション）を自動非表示
  - 不要なバナー・ヘッダーを非表示
  - ポストヘッダーのレイアウト最適化
  - ポスト日時を絶対時刻で表示

- **設定管理**
  - 設定のエクスポート/インポート（JSON形式）
  - 設定のクリア

- **フィルタ機能**
  - 任意の正規表現にマッチするポストを非表示
 

## 使用方法

# インストール

（以下は一例です。使用するブラウザに応じた拡張機能のインストールを実行してください。）

1. Releases から zip をダウンロードする
1. ブラウザで拡張機能の設定画面を開く
2. 「デベロッパーモード」を有効にする
3. 「(from .zip/.crx/.user.js) をタップし、ダウンロードした zip を選択する

# 使用

1. https://x.com/run-twitama-modoki にアクセスする
2. （ツイタマモドキが自動的に起動する）
3. 設定カラムから新規カラムを追加する
4. 各カラムで自由にページ移動する（ホーム、検索、リスト、コミュニティ、通知、DMのURLが自動保存される）

## 開発情報

### 前提条件

- Node.js (v18以上)
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# ビルド
npm run build
```

### Chrome拡張機能として読み込む

1. Chromeで拡張機能の設定画面（`chrome://extensions/`）を開く
2. 「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリックする
4. `dist` フォルダを選択する

### 技術スタック

- **TypeScript** - 型安全性
- **React** - UIフレームワーク
- **Vite** - ビルドツール
- **CRXJS** - Chrome拡張機能ビルド
- **Swiper** - スワイプ操作
- **HammerJS** - ジェスチャー認識
- **Zustand** - 状態管理

### プロジェクト構造

```
twitama-modoki/
├── src/
│   ├── components/      # Reactコンポーネント
│   │   ├── columns/     # カラムコンポーネント
│   │   ├── App.tsx      # メインアプリ
│   │   ├── ColumnSwiper.tsx  # スワイプ機能
│   │   ├── TabBar.tsx   # タブバー
│   │   └── RateLimitPanel.tsx  # レート制限表示
│   ├── types/           # TypeScript型定義
│   ├── storage/         # chrome.storage管理
│   ├── store/           # Zustand状態管理
│   ├── hooks/           # カスタムフック
│   ├── config/          # 設定・セレクタ
│   ├── content/         # Content Script
│   ├── background/      # Background Script
│   └── styles/          # CSS
├── public/              # 静的アセット
├── scripts/             # ビルドスクリプト
└── dist/                # ビルド出力
```
